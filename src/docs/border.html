Border

Border utilities to easily change border color and width, as well as make them responsive.
Bootstrap documentation

Responsive
  • .border-{breakpoint}, .border-top-{breakpoint}, .border-right-{breakpoint}, .border-bottom-{breakpoint}, .border-left-{breakpoint} - adds a border at the specified and subsequent breakpoints.
  • .border-{breakpoint}-0, .border-top-{breakpoint}-0, .border-right-{breakpoint}-0, .border-bottom-{breakpoint}-0, .border-left-{breakpoint}-0 - removes a border at the specified and subsequent breakpoints.
Colors
  • .border-gray-700 - changes the border color to $gray-700.
Width
  • .border-2 - doubles the default border width.
